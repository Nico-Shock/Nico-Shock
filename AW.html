My Arch Workspace Setup

This guide outlines the steps to set up an optimal Arch Linux work environment.

Preparation

Update System

Ensure your system is up-to-date. 

sudo pacman -Syu

Install Basic Packages

Install essential development tools and utilities. 

sudo pacman -S base-devel git vim neofetch htop --noconfirm

Desktop Environment (Example: GNOME)

Install GNOME

Install the GNOME desktop environment and related packages. 

sudo pacman -S gnome gnome-extra gdm --noconfirm

Enable GNOME Display Manager (GDM)

Enable GDM to load GNOME on startup. 

sudo systemctl enable gdm.service --now

Productivity Software

Web Browser (Firefox)

Installation of the Firefox web browser. 

sudo pacman -S firefox --noconfirm

Text Editor (VS Code)

Installation of Visual Studio Code. 

sudo pacman -S visual-studio-code-bin --noconfirm

(Note: visual-studio-code-bin is from the AUR and requires an AUR helper like yay or paru)

Terminal Emulator (Alacritty)

Installation of a fast GPU-accelerated terminal emulator. 

sudo pacman -S alacritty --noconfirm

Additional Configurations

Dotfiles

Manage your configuration files (Dotfiles) with Git. 

git clone <your-dotfiles-repo-url> ~/.dotfiles
# Create symlinks or run script

Fonts

Install useful fonts for a better terminal and code view. 

sudo pacman -S ttf-fira-code ttf-jetbrains-mono --noconfirm

Theming

Customize your GNOME theme with GNOME Tweaks. 

sudo pacman -S gnome-tweaks --noconfirm

Conclusion

After completing all steps, restart your system to apply the changes. 

sudo reboot
